    Installation

    Installation

    Measurements.jl is available for Julia 0.7 and later versions, and can be installed with Julia built-in package manager. In a Julia session run the commands

    pkg> update
pkg> add Measurements

    Older versions are also available for Julia 0.4-0.6.
